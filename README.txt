Student Database Application
	This Java application interacts with a PostgreSQL database to perform CRUD (Create, Read, Update, Delete)  	operations on a "students" table.

Instructions

	1. Clone the Repository: Clone the repository from Git Hub to your local machine

	2. Set Up PostgreSQL Database: Make sure you have PostgreSQL installed on your system. Create a database 	named assignment3 and you can accurately create a table and insert the inital data using the script.sql found 	in Q1_DatabaseScript derictory.

	3. Java Application: There is two java files...
		Student.java is the file that have all the functions and main function.
		DatabaseConnection.java is the file where the connections of the database and the application is done  
How to Run
	Run the Application using the Student.java.

Dependencies
	Java Development 
	PostgreSQL JDBC
        PostgreSQL Database
